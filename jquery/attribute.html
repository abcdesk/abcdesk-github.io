jQuery Attribute属性值

addClass向被选元素添加一个或多个类

<script>
  $("th,td").addClass("text-center");
</script>

attr() 方法设置或返回被选元素的属性值

<a href="aaaaaa" id='hh'></a>

<script>
  var p = $('#hh').attr('href');
  console.log(p);
</script>

hasClass() 方法检查被选元素是否包含指定的 class

$(selector).hasClass(class);
alert($("p:first").hasClass("intro"));

removeAttr() 方法从被选元素中移除属性

$(selector).removeAttr(attribute)

removeClass() 方法从被选元素移除一个或多个类

$(selector).removeClass(class);

toggleClass对被选元素进行添加/删除类的切换操作

<script>
  $('[data-toggle="offcanvas"]').click(function() {
    $('.row-offcanvas').toggleClass('active')
  });
</script>

css设置CSS属性

<script>
  $("#content>ul>li").css({"padding":"10px 5px","color":"#7d8688"});
</script>

html() 方法返回或设置被选元素的内容 (inner HTML)

$(selector).html();         //返回元素内容
$(selector).html(content);  //设置元素内容
$("p").html("Hello world!");

标签赋初值

自闭合标签赋初值用$().val()，比如：<input type="text" name="text" value="123">

闭合标签赋初值用$().text()，比如：<textarea id="id_textarea"></textarea>

# 给textarea标签赋初值
$('#id_textarea').text($('#form_text').attr('my_text'));

// 给单选 赋初值
$("#id_0").attr("checked", $('#id_radio').attr('gender') == '男' ? true : false);
$("#id_1").attr("checked", $('#id_radio').attr('gender') == '女' ? true : false);
$("#id_2").attr("checked", $('#id_radio').attr('gender') == '未知' ? true : false);
form.render(); //更新全部

input框赋值

<input type="text" name="number" id="number" value=""/>
#通过type赋值
$(":text").val("100");

#通过id赋值
$("#number").val("123");
$("#number").attr("value","123");

#通过name赋值
$("input[name='number']").val("123");
$("input[name='number']").attr("value","123");

获取data-*属性值

<li id="myId" data-id="122" data-vice-id="11">获取id</li>

使用data()方法

var id = $("#myId").data("id"); //122
var viceId = $("#myId").data("vice-id"); //11
//赋值
$("#myId").data("id","100");//100

使用attr()方法

var id = $("#myId").attr("data-id"); //122
var viceId = $("#myId").attr("data-vice-id"); //11
//赋值
$("#myId").attr("data-id","100");//100
注意！！

data()的值进行修改并不会影响到DOM元素上的data-*属性的改变。
