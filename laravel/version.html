Laravel 版本选择

laravel有两种版本类型：LTS版本和一般发行版。 LTS 版本：长期支持版本，英文 Long Term Support 的缩写，提供两年的 Bug 修复支持，三年的安全修复支持。 一般发行版：只提供6个月的Bug修复支持，一年的安全修复支持。 如何选择呢？ （1）如果是商业项目开发，应该优先考虑LTS版本，因为它的安全性和稳定性，不应该 使用最新版本的Laravel 一般发行版； （2）如果是自己搭建项目练手，建议优先考虑最新的发行版本，因为你可以优先体验最新的技术和扩展之类的。

查看Laravel版本号的三种方法

1.php artisan

$ php artisan --version
$ php artisan -V

2.查看Application.php文件

$ vim vendor/laravel/framework/src/Illuminate/Foundation/Application.php

3.写在路由里

Route::get('laravel-version', function(){
    $laravel = app();
    return "Your Laravel version is ".$laravel::VERSION;
});
