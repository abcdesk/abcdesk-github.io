原型项目

  • Laravel 5 Boilerplate —— 基于当前Laravel最新版本（Laravel 6.0）并集成Boilerplate的项目

    • 详情

    count(): Parameter must be an array or an object that implements Countable

    分析：查询官方手册发现7.2版本的php有更新，count传入非数组的变量会出现warning；本代码中query->wheres 为 NULL，count(NULL)则报错。

    解决方案:

    定位到出错文件/vendor/laravel/framework/src/Illuminate/Database/Eloquent/Builder.php

        // 1231行 $originalWhereCount = count($query->wheres);修改如下：
    if ($query->wheres != null) {
        $originalWhereCount = count($query->wheres);
    } else {
        $originalWhereCount = 0;
    }
  • Element In Laravel —— 基于 Laravel 5.7 + Vue2 + Element UI 的原型项目（长期未更新）
    • composer install一直报错，无法完成

    解决方案:

    1. 修改根目录composer.json文件：

      2. "laravel/framework": "5.7.*",=>"laravel/framework": "5.5.*",

    2. 删除根目录composer.lock文件，重新运行composer install
