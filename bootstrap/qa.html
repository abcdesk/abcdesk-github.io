Bootstrap QA问题及解决方案

如何让pre标签自动换行？

解决方案: 

pre {
    white-space: pre-wrap;
    word-wrap: break-word;
}
如何去除Bootstrap阴影、边框，并让背景透明？

解决方案:使用CSS 

.glyphicon {
  /*color: coral;*/
  font-size: 20px;
  color: red;
  background-color:rgba(0,0,0,0);/*背景透明*/
  background: #000 9;
  /*CSS Hack,只能对ie9以下浏览器ie6,ie7,ie8有效，否侧ie10,FF,Cherome会失去透明效果*/
  filter: Alpha(opacity=60);
  /*只对ie7,ie8有效*/
}

box-shadow: none; /*去除阴影*/
border: none; /*去除边框*/
