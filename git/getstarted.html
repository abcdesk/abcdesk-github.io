Git 开工准备

安装Git

下载Git 然后，解压缩到指定目录下，配置好环境变量即可完成安装。

配置环境变量

桌面计算机图标上单击鼠标右键->属性->高级系统设置->高级->环境变量

GIT_HOME = "C:\sde\PortableGit"
Path = %GIT_HOME%;...

安装完成后，还需要最后一步设置，在命令行输入：

$ git config --global user.name "Your Name"
$ git config --global user.email "email@example.com"

查看配置信息

$ git config --system --list    #查看系统config
$ git config --global --list    #查看当前用户（global）配置
$ git config --local  --list    #查看当前仓库配置信息

生成 SSH 公钥

默认情况下，用户的 SSH 密钥存储在其 ~/.ssh 目录下。

$ cd ~/.ssh
$ ssh-keygen

首先 ssh-keygen 会确认密钥的存储位置（默认是 .ssh/id_rsa），然后它会要求你输入两次密钥口令。如果你不想在使用密钥时输入口令，将其留空即可。 然后我们会看到一对以 id_dsa 或 id_rsa 命名的文件，其中一个带有 .pub 扩展名。 .pub 文件是你的公钥，另一个则是私钥。

右键添加 Git Bash Here 菜单

将下列代码保存为.reg文件，然后双击运行添加到注册表，即可实现。

Windows Registry Editor Version 5.00
; Open files
; Default Git-Bash Location C:\Program Files\Git\git-bash.exe

[HKEY_CLASSES_ROOT\*\shell\Open Git Bash]
@="Open Git Bash"
"Icon"="C:\\sde\\PortableGit\\git-bash.exe"

[HKEY_CLASSES_ROOT\*\shell\Open Git Bash\command]
@="\"C:\\sde\\PortableGit\\git-bash.exe\" \"--cd=%1\""

; This will make it appear when you right click ON a folder
; The "Icon" line can be removed if you don't want the icon to appear

[HKEY_CLASSES_ROOT\Directory\shell\bash]
@="Open Git Bash"
"Icon"="C:\\sde\\PortableGit\\git-bash.exe"


[HKEY_CLASSES_ROOT\Directory\shell\bash\command]
@="\"C:\\sde\\PortableGit\\git-bash.exe\" \"--cd=%1\""

; This will make it appear when you right click INSIDE a folder
; The "Icon" line can be removed if you don't want the icon to appear

[HKEY_CLASSES_ROOT\Directory\Background\shell\bash]
@="Open Git Bash"
"Icon"="C:\\sde\\PortableGit\\git-bash.exe"

[HKEY_CLASSES_ROOT\Directory\Background\shell\bash\command]
@="\"C:\\sde\\PortableGit\\git-bash.exe\" \"--cd=%v.\""
