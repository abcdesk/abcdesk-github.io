Git Tag标签

标签

$ git tag v1.0  #创建标签v1.0

$ git log --pretty=oneline --abbrev-commit
$ git tag v0.9 f52c633  #对commit id是f52c633打上标签v0.9

$ git tag -a <tagname> -m "blablabla..."  #可以指定标签信息

$ git tag               #显示所有的tag
$ git tag -l 'v1.0.*'   #查看某个版本系列的tag
$ git show v1.0         #查看标签的详情，可以看到你commit的内容

$ git push origin v1.0  #推送标签
$ git tag -d v1.0       #删除本地
$ git push origin :refs/tags/v1.0 #删除远程
