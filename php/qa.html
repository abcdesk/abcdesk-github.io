PHP QA问题及解决方案

判断变量为空

解决方案:

isset
判断变量是否被初始化
empty
检测变量是否为"空"(TRUE:"假"、"空"、"0"、"NULL"、"未初始化")
var == null
判断变量是否为"空"(TRUE:"假"、"空"、"0"、"NULL")
is_null
检测变量是否为"null"(TRUE:"NULL")
var === null
检测变量是否为"null"(TRUE:"NULL")

以isset为例举例说明：

if (!isset($a)) {
    echo '未被初始化';
} else {
    echo '已初始化';
}
判断两个字符串是否相等

解决方案:

使用函数strcmp

if (strcmp($var1, $var2) !== 0) {
    echo '$var1 is not equal to $var2 in a case sensitive string comparison';
}
Memory limit errors

PHP Fatal error: Allowed memory size of XXXXXX bytes exhausted <...>

分析

memory_limit设置太大会影响系统速度，因为系统和数据库及其他程序同样需要内存空间，一般系统和数据库内存空间是自己分配的。

解决方案:

修改PHP的memory_limit限制3种方法

  1. 在php.ini中，找到“memory_limit”这一项，如果没有，你可以在文件的尾部自己增加这个参数。

    2. memory_limit = 128M;

  2. 网站的根目录下“.htaccess”文件

    3. php_value memory_limit 128M;

  3. 在你的php代码中增加以下命令行

    4. ini_set('memory_limit','128M');

附注：查看php内存设置

php -r "echo ini_get('memory_limit').PHP_EOL;"
