Tool Sublime Text 3

1.安装Sublime Text 3

下载Sublime Text Build 3114 解压到指定目录即可完成安装。

基本配置，单击菜单栏Preferences→Setting - User

{
    "default_line_ending": "unix",
    "expand_tabs_on_save": true,
    "font_size": 16,
    "ignored_packages":
    [
        "Vintage"
    ],
    "show_encoding": true,
    "show_line_endings": true,
    "tab_size": 4,
    "translate_tabs_to_spaces": true,
    "trim_trailing_white_space_on_save": true,
    "update_check": false,
    "word_wrap": false
}

2.安装Package Control

The simplest method of installation is through the Sublime Text console. The console is accessed via the ctrl+` shortcut or the View > Show Console menu. Once open, paste the appropriate Python code for your version of Sublime Text into the console. 

import urllib.request,os,hashlib; h = '6f4c264a24d933ce70df5dedcf1dcaee' + 'ebe013ee18cced0ef93d5f746d80ef60'; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); by = urllib.request.urlopen( 'http://packagecontrol.io/' + pf.replace(' ', '%20')).read(); dh = hashlib.sha256(by).hexdigest(); print('Error validating download (got %s instead of %s), please try manual install' % (dh, h)) if dh != h else open(os.path.join( ipp, pf), 'wb' ).write(by)

使用Ctrl+Shift+P打开命令行模式，在里面输入Install Package即可搜索需要的Package。

Sublime常用快捷键

序号 快捷键 功能
1 Ctrl + Shift + P 打开命令行模式
2 Ctrl + D 选中光标所占的文本，继续操作则会选中下一个相同的文本。
3 Alt + F3 一次性选择全部的相同文本。
4 Ctrl + L 选中整行，继续操作则继续选择下一行。
5 Ctrl + Shift + L 先选中多行，再按下快捷键，会在每行行尾插入光标，即可同时编辑这些行。
6 Ctrl + Enter 在下一行插入新行。
7 Ctrl + Shift + Enter 在上一行插入新行。
8 Ctrl + M 光标移动至括号内结束或开始的位置。
9 Ctrl + J 合并选中的多行代码为一行。
10 Ctrl + Shift + D 复制光标所在整行，插入到下一行。
11 Ctrl + K + K 从光标处开始删除代码至行尾。
12 Ctrl + Shift + K 删除整行。
13 Tab 向右缩进。
14 Shift + Tab 向左缩进。
15 Ctrl + K + U 转换大写。
16 Ctrl + K + L 转换小写。
17 Ctrl + / 注释单行。
18 Ctrl + Shift + / 注释多行。
19 F11 全屏模式
20 Shift + F11 免打扰模式

问题及解决方案

Sublime Text 3无法安装插件

解决方案:

菜单Preferences→Package Settings→Package Control→Setting - User

"channels":
[
"http://static.bolin.site/channel_v3.json",
//"https://packagecontrol.io/channel_v3.json",
//"https://web.archive.org/web/20160103232808/https://packagecontrol.io/channel_v3.json",
//"https://gist.githubusercontent.com/nick1m/660ed046a096dae0b0ab/raw/e6e9e23a0bb48b44537f61025fbc359f8d586eb4/channel_v3.json"
],
如何将Windows文件的换行符转换为Unix风格？

解决方案:

菜单View→Line Endings→Unix 然后保存文件。

如何用HTML-CSS-JS Prettify格式化Laravel的blade模板文件？

解决方案:

菜单Preference→Package Settings→HTML-CSS-JS Prettify→Plugin Options - Default 找到allowed_file_extensions添加"blade.php"，然后保存文件。 

"allowed_file_extensions": ["htm", "html", "xhtml", "shtml", "xml", "svg", "vue", "blade.php"],
