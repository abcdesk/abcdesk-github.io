Tool Emmet插件的使用

生成html5模板

新建一个HTML文档，输入“!”或“html:5”，然后按Tab键(或Ctrl+E)

嵌套运算符(Nesting operators)

运算符 名称 作用 举例
> 子代操作符(Child) 生成后代元素 nav>ul>li
+ 兄弟操作符(Sibling) 生成兄弟元素 div+p+bq
^ 返回上级操作符(Climb-up) 生成上级元素 div+div>p>span+em^bq
* 乘法操作符(Multiplication) 重复生成多份 ul>li*5
() 分组操作符(Grouping) 生成分组 (header>h1+div>p)+(main>p)+(footer>ul>li*3)

属性操作符(Attribute Operators)

操作符 名称 作用 举例
# ID 生成ID #container或div#container
. CLASS 生成类名 .container或div.container
[] 自定义属性(Custom attributes) 生成自定义属性 td[rowspan=2 colspan=3 title="test"]
$ 项目编号(Item numbering) 对生成内容依次编号 ul>li.item$*5
@- @-修饰符(Modifier) 更改编号顺序(倒序) ul>li.item$@-*5
@N @N修饰符(Modifier) 指定开始的序号 ul>li.item$@3*5
{} 文本操作符(Text) 生成文本内容 a{click me}
注意！！

1.可以在文本中的任意位置键入和展开缩写 2.缩写中不能包含空格，因为空格是一个停止符号，Emmet停止缩写解析。如下： (header > ul.nav > li*5) + footer

HTML

All unknown abbreviations will be transformed to tag, e.g. foo → <foo></foo>.
